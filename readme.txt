This plugin adds a button to the admin panel that links to the create a new blog page. button disapears after user has created one blog. 

== Description ==

New users often do not know where to start when it with Wordpress. This plugin places a bright yellow button at the top of the dashboard to guide the user to the place to create their first blog. After the user has made a blog the button dissapears. This plugin is configured to run in Wordpress network. Plugin may need modification to run on standalone install of wordpress.



== Installation ==

1. Upload `create_new_blog_notice.php` to the `/wp-content/mu-plugins/` directory
